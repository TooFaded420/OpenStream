# OpenClaw Stream Deck - Distribution Packages

## 📦 Package Options

### Option 1: Home Assistant Addon (Recommended for HA users)
**Best for:** Home Assistant users, smart home enthusiasts

**Installation:**
1. Add custom repository to Home Assistant
2. Install "OpenClaw Stream Deck" addon
3. Configure gateway URL
4. Start addon

**Features:**
- Integrates with Home Assistant dashboard
- Web UI for configuration
- Auto-discovery of OpenClaw
- Works with any Stream Deck

---

### Option 2: Standard Package (Recommended for everyone)
**Best for:** Quick setup, non-Home Assistant users

**Installation:**
1. Download `OpenClaw-Standard-v1.0.zip`
2. Extract and run `INSTALL.bat`
3. Restart Stream Deck
4. Add OpenClaw actions to your buttons

**Features:**
- Works immediately
- No dependencies
- 15 built-in actions
- Custom icons included

---

### Option 3: Custom Build (For developers)
**Best for:** Modifying code, adding features

**Installation:**
1. Clone repo
2. Edit files in `plugin/` folder
3. Run `build.bat` to package
4. Install manually
