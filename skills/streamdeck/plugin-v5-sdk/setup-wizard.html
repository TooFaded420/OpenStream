OpenClaw Stream Deck Setup

This plugin is installed and the Setup key has already synced auth token to your Stream Deck gateway config.

Quick Verify

  1. In Stream Deck app, switch to profile OpenClaw Plus v5.
  2. Press Status. You should see a short success response.
  3. Press Search or Spawn. These should trigger a gateway run.
  4. Press Nodes and Agents for counts.

Status Codes

  • OPEN: Setup page opened successfully.
  • SYNC: Gateway token was re-synced from ~/.openclaw/openclaw.json.
  • NOCFG: Missing OpenClaw config file.
  • NOTOK: OpenClaw config has no gateway auth token.

Manual Fallback

From terminal:

cd C:\Users\jrlop\.openclaw\workspace\skills\streamdeck

powershell -ExecutionPolicy Bypass -File .\scripts\install-v5-sdk.ps1 -Force
