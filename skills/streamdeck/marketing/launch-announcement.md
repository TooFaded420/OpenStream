**Title:** OpenClaw Stream Deck Plugin - Control Your AI from Physical Buttons

**Body:**

🎉 **NEW:** Transform your Stream Deck into an AI command center!

After months of development, I'm thrilled to launch the **OpenClaw Stream Deck Plugin** - the ultimate way to control your AI assistant without typing a single command.

### ✨ What It Does

**25+ Built-in Actions:**
- 🎙️ Toggle TTS (voice responses)
- 🤖 Spawn AI sub-agents
- 📊 Check system status
- 🧠 Switch AI models
- 💻 Spawn coding agents
- 🔍 Web search
- 🧠 Memory search
- 📤 Send messages
- 🔄 And 17 more!

**Key Features:**
- ✅ Works with ALL Stream Deck models (Mini, MK.2, Plus, XL, Pedal)
- ✅ Multi-gateway support (control multiple OpenClaw instances)
- ✅ Dynamic status display (keys show live status)
- ✅ Custom coral-red icons
- ✅ Easy setup - works in 60 seconds

### 🎯 Perfect For

- **Developers:** Spawn coding agents while you work
- **Streamers:** AI co-hosts, voice alerts, chat automation
- **Power Users:** Quick access to all OpenClaw features
- **Accessibility:** Physical buttons instead of typing

### 💰 Pricing

**Free Version:** Core actions (TTS, Status, Spawn)
