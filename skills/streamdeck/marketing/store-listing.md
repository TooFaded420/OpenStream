# OpenClaw Stream Deck Plugin - Store Listing

## 🎯 App Title
**OpenClaw AI Control**

## 📝 Short Description (50 chars)
Control your AI assistant from Stream Deck

## 📄 Full Description
Transform your Stream Deck into an AI command center!

**OpenClaw AI Control** brings the power of OpenClaw directly to your fingertips. Spawn AI agents, toggle voice responses, check system status, and control your entire OpenClaw setup - all without leaving your current window.

### ✨ Features

**🎙️ Voice Control**
- Toggle TTS (text-to-speech) instantly
- Send voice messages to your AI
- Adjust voice settings on the fly

**🤖 Agent Management**
- Spawn sub-agents for parallel tasks
- Monitor active agents
- Kill agents with one tap

**⚡ Quick Actions**
- Check OpenClaw status at a glance
- Switch between AI models
- Search web and memory
- Restart gateway when needed

**🎨 Beautiful Design**
- Custom coral-red icons
- Status feedback on keys
- Works with all Stream Deck models

### 🖥️ Supported Devices
- Stream Deck Mini (6 keys)
- Stream Deck MK.1/MK.2 (15 keys)
- Stream Deck Plus (15 keys + 4 dials)
- Stream Deck XL (32 keys)
- Stream Deck Pedal (3 pedals)

### 🚀 Quick Setup
1. Install this plugin
2. Enter your OpenClaw gateway URL
3. Start controlling your AI!

### 💡 Use Cases
- **Developers**: Spawn coding agents while you work
- **Streamers**: TTS alerts and AI co-hosts
- **Power Users**: Quick access to all OpenClaw features
- **Accessibility**: Physical buttons for AI control

### 🔗 Requirements
- Windows 10/11
- OpenClaw installed and running
- Stream Deck software
